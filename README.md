# VK бот для магазина "Молоток"

Простой бот для ВК на `Python + vkbottle + SQLite`.

## Возможности

- Каталог: молотки, гвозди, перчатки
- Цена и описание товара
- Оформление заявки: имя, телефон, товар, количество
- Передача заказа менеджеру в ЛС
- Защита от дублей заказа (за последние 24 часа)
- Экспорт заказов в CSV (команда `экспорт` для менеджера)

## Установка (Windows)

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка

1. Создай файл `.env` по примеру `.env.example`.
2. Вставь токен сообщества и `MANAGER_ID`.

```env
VK_BOT_TOKEN=твой_токен
MANAGER_ID=123456789
```

## Запуск

```powershell
python bot.py
```

## Команды бота

- `Начать` / `Меню`
- `Каталог`
- `Заказать`
- `Помощь`
- `Отмена`
- `Экспорт` (только менеджер)
